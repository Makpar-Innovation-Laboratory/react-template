Class ReplyComponent

ReplyComponent

Description

A widget for replying to comments. Pass in a commentId to bind this component to a particular Comment. Initially, when showEditor is false, ReplyComponent will display a mat-card with a reply mat-icon button. If the user clicks on the reply button, the state of showEditor will switch to true and a commentForm will unfold to allow the user to reply to this Comment.

Example Usage

 
<app-reply [commentId]="1" [tooltip]="Reply to a comment!"></app-reply>
